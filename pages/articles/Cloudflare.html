Internet Great Good Samaritan: Cloudflare

Jun 3 2025

In passage "My Website, My Rules", I said that I used to host my website on a Vultr server for $5 per month, which was quite expensive. then, I used GitHub Pages since it's free and worked well, but due to the Great Firewall (GFW) in China, I couldn't access it without a VPN.

Later, while exploring internal network penetration for my Home Assistant system, I discovered Cloudflare. It's a powerful tool that offers services like CDN, website hosting, and security features—all for free.

Since Cloudflare's hosting isn't affected by the GFW and is completely free, I switched to it. Now, I only need to pay for the domain, which costs around 70 RMB per year. This is much cheaper than using Vultr.

Cloudflare also provides many useful services for developers and is truly a great contributor to the internet.

2025年6月3日

Cloudflare 还为开发者提供了许多实用的服务，是真正的互联网大善人。
