My Website, My Rules

Apr 23 2025

Hello, I'm Junxi Bao

This is the very first passage on this website. In this post, I'll share the process of how I built this site and give a brief introduction to what it's all about.

Let me start with the purpose behind creating this website. Recently, I've been learning how to build websites and have helped a few of my classmates create sites for different purposes. That made me wonder—why not build one for myself? I thought it would be really cool to have a personal site where I could share some of my thoughts and diaries.

Some people might say my site looks fancy or beautiful, but to be honest, all the CSS was written by ChatGPT—because I can't write CSS at all! Thanks to AI, I was able to build this website quite efficiently. I only had to write the HTML and some JavaScript, which I find much easier. After that, I rented a server in Tokyo for $5 a month to host the site—and that's pretty much it!

This website is divided into four main sections: About Me, Articles, GitHub, and Email Me.

  • About Me: a short introduction about myself.

  • Passages: diary-style posts that I write in my spare time.

  • GitHub: a link to my GitHub profile, where I share many of my projects.

  • Email Me: a form to send me messages directly.
