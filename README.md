# JunxiBao's Terminal - 极客风格个人网站

## 🚀 项目概述

这是一个完全重新设计的极客风格个人网站，采用深色模式、终端风格设计，具有现代化的UI和丰富的交互效果。

## ✨ 主要特性

### 🎨 视觉设计
- **极客风格**: 采用黑客/程序员风格的深色主题
- **深色模式**: 护眼的深色配色方案，绿色终端风格
- **Matrix雨滴**: 动态的Matrix风格背景效果
- **终端窗口**: 模拟真实终端界面的设计元素
- **霓虹发光**: 绿色霓虹灯效果的按钮和文字

### 📱 响应式设计
- **移动端友好**: 完全适配各种屏幕尺寸
- **折叠菜单**: 移动端左上角折叠式导航菜单
- **触摸优化**: 针对触摸设备优化的交互体验

### 🎭 交互效果
- **滚动动画**: 平滑的滚动动画和视差效果
- **悬停效果**: 丰富的悬停动画和视觉反馈
- **粒子系统**: 动态粒子背景效果
- **故障效果**: 文字故障动画效果
- **波纹效果**: 按钮点击波纹动画

### 🏗️ 技术架构
- **纯前端**: 使用HTML5、CSS3、JavaScript构建
- **模块化**: 组件化的JavaScript架构
- **性能优化**: 优化的动画和交互性能
- **兼容性**: 支持现代浏览器

## 🎨 设计系统

### 颜色方案
- **主背景**: `#0a0a0a` (深黑色)
- **次要背景**: `#1a1a1a` (深灰色)
- **主要文字**: `#00ff41` (矩阵绿)
- **次要文字**: `#00cc33` (深绿色)
- **强调色**: `#0080ff` (蓝色), `#8000ff` (紫色)

### 字体系统
- **主要字体**: JetBrains Mono, Fira Code, Consolas
- **等宽字体**: 保持代码和终端的视觉一致性
- **字体大小**: 响应式字体大小系统

### 组件库
- **按钮**: 霓虹发光效果，悬停动画
- **卡片**: 悬停提升，边框发光
- **导航**: 粘性导航，滚动效果
- **终端**: 真实终端窗口模拟

## 🚀 使用方法

### 1. 本地运行
```bash
# 克隆项目
git clone [repository-url]
cd web

# 使用本地服务器运行（推荐）
python -m http.server 8000
# 或
npx serve .

# 在浏览器中访问
http://localhost:8000
```

### 2. 自定义内容
- **修改个人信息**: 编辑 `src/home.html` 中的个人介绍
- **添加文章**: 在 `src/articles/` 目录下创建新文章
- **更新文章列表**: 修改 `src/passage.html` 中的文章卡片
- **自定义样式**: 编辑 `static/css/geek-theme.css`

### 3. 部署
- **GitHub Pages**: 直接推送到GitHub仓库
- **Netlify**: 拖拽部署
- **Vercel**: 自动部署
- **传统服务器**: 上传到Web服务器
